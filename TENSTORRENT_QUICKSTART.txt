BarraCUDA Tenstorrent Quickstart
================================

Build the compiler (needs gcc, nothing else):

    make

Compile a CUDA kernel to Metalium C++:

    ./barracuda --tensix tests/vector_add.cu -o compute_kernel.cpp

This outputs a C++ file containing encoded SFPI instructions targeting Blackhole B0.
The output uses TT-Metalium's tile_regs and circular buffer APIs, so it slots into
a standard TT-Metal project as the compute kernel.

To run on hardware you need TT-Metalium installed and a working Blackhole device.
Wrap the generated compute_kernel.cpp in a TT-Metal host program that sets up
circular buffers, dispatches reader/writer/compute, and launches.

The backend is new and has not been validated on hardware yet. If you have a working
Blackhole and want to try it, we would love to hear from you.
